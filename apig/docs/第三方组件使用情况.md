# MVP 第三方组件使用情况

## 1. 网关与微服务基础组件

- `Spring Boot`：各微服务基础运行框架。  
- `Spring Cloud Gateway`：`gateway-dataplane` 网关内核。  
- `Spring Security` + `OAuth2 Resource Server`：统一认证鉴权。  
- `jjwt`：JWT 签发与校验。  
- `Resilience4j`：超时、重试、熔断。  
- `springdoc-openapi`：OpenAPI 文档生成。  

## 2. 中间件与基础设施

- `Nacos`：服务注册发现 + 配置中心。  
- `MySQL`：业务元数据存储（API、路由、订阅、凭证等）。  
- `Redis`：缓存、限流计数、防重放窗口。  
- `RocketMQ`：配置发布事件、审计异步消息。  

## 3. 可观测组件

- `Micrometer`：应用指标采集。  
- `Prometheus`：指标抓取存储。  
- `Grafana`：监控看板。  
- `OpenTelemetry`：链路埋点标准。  
- `Loki`（或 ELK）：日志检索。  

## 4. 容器化与交付

- `Docker`：镜像打包与运行。  
- `Docker Compose`：本地一体化联调部署。  
- `Maven`：Java 项目构建。  

## 5. 组件与微服务映射

- `gateway-dataplane`：Spring Cloud Gateway、Spring Security、Resilience4j、Redis、Nacos。  
- `gateway-controlplane`：Spring Boot、Nacos、MySQL、RocketMQ。  
- `api-management`：Spring Boot、MySQL、Nacos。  
- `auth-service`：Spring Security、OAuth2、jjwt、MySQL、Redis。  
- `app-portal-service`：Spring Boot、MySQL、Nacos。  
- `observability-service`：Micrometer、Prometheus、Grafana、Loki（或 ELK）。
